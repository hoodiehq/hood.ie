Report a bug

  1. Please check our FAQ. We have provided a list of known bugs, please check if yours is in there.
  2. If it’s not in this list, please use this form to file the bug. Your detailed error message and description help us solve the bug and helps us make this content available for other Hoodie users as well. Please disclose security vulnerabilities responsibly and contact us directly.
  3. If you have general questions or need help, look at this page for finding answers.
